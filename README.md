# SchuleWebsite

Dies ist eine moderne Website für die Schule, basierend auf dem gegebenen Konzept.

## Neue Features:
- **Cooles Design**: Gradienten, Animationen, Hover-Effekte, Responsive.
- **Dark Mode**: Toggle-Button im Header, gespeichert in localStorage.
- **Interaktivität**: Smooth Scrolling, Animation on Scroll, verbesserte Sidebar.
- **Mehr Unterseiten**: Schulleitung, Daten, Galerie, Veranstaltungen, Kultur, Karriere.
- **Barrierefreiheit**: Hoher Kontrast, Tastatur-Navigation.

## Struktur
- `index.html`: Startseite
- `css/styles.css`: Modernes Stylesheet mit CSS-Variablen
- `js/script.js`: JavaScript für Funktionen
- `pages/`: Mehr Unterseiten
- `images/`: Bilder (fügen Sie echte hinzu)
- `downloads/`: PDFs (fügen Sie echte hinzu)

## Wie starten
Öffnen Sie `index.html` in einem Browser oder verwenden Sie den lokalen Server: http://localhost:8000

## Anpassungen
- Ersetzen Sie Platzhalter-Bilder und Inhalte.
- Fügen Sie echte Daten hinzu.
